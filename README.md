# DSA Chatbot 🤖

A full-stack AI chatbot that answers Data Structures and Algorithms questions.

## Tech Stack
- React
- Node.js (Express)
- Gemini API

## Features
- ChatGPT-style UI
- DSA-focused responses
- Rude replies for non-DSA questions 😈

## Run locally
### Backend
node server.js

### Frontend
cd client
npm start
